## ğŸ”§ How It Works

### ATS Analysis
The analyzer checks your resume for:
- **Keywords** - Technical skills, action verbs, and soft skills
- **Sections** - Contact info, summary, experience, education, skills
- **Formatting** - Length, bullet points, quantifiable achievements
- **Readability** - Sentence length and word complexity

### Scoring
- 80-100: Excellent
- 60-79: Good
- 40-59: Fair
- 0-39: Needs Improvement
